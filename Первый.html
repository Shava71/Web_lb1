Язык HTML

HTML (HyperText Markup Language) — стандартный язык разметки документов для создания веб-страниц и веб-приложений.

Он используется для структурирования информации и создания гипертекстовых ссылок между документами.

HTML является основным строительным блоком для всех веб-сайтов, включая текст, изображения и интерактивные элементы.
